# DataForge

**DataForge** is an open, real-time data engineering platform that ingests,
processes, stores, and serves analytics-ready data using industry-standard
tools and architectures.

## Project Goals
- Build batch and real-time data pipelines
- Implement data lake and data warehouse architectures
- Enable analytics through BI tools and APIs
- Follow production-grade engineering practices

## Tech Stack
- Python
- Apache Kafka
- Apache Spark
- Apache Airflow
- PostgreSQL
- MongoDB
- MinIO
- Docker
- Power BI
